CRISPRpred

CRISPRpred is based on the pipeline presented in Rahman et al. (2017). It was retrained with the dataset of DeepCRISPR. This pipepline incorporates both position specific and position independent features. The features are ranked using Random Forest algorithm. Finally a SVM model was trained using the selected features.
